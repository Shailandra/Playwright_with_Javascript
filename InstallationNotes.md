Playwright Prerequisites
------------------------------
1. install node 
2. install npm
3. install VS Code
4. install playwright
5. create project folder --> open in VS code
6. install playwright using terminal
     npm init playwright@latest
     package.json -- node project management file
     playwright.configuration
     tests -- we can wright all the playwright tests
7. npm playwright -v
8. npx playwright 
    test to run the playwright test
9. npx playwright test --headed
10. npx playwright show-report

11. We can generate test case in two Ways 
    a. By code Gen App
    b. by writing out own Script
